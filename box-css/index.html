Anchor based CSS/jQuery box

It is possible to swap the content with pure CSS anchor links as well, directing the href into a div with #ID. That approach however makes the box jump up on click in case the target div is not placed at the absolute top of the page.
Hence jQuery. Window #hash makes it then possible to link to a specific content.

Will be compatible with major CMS systems (not with TYPO3)
